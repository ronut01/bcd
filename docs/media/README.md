# README Media

This folder contains lightweight visual assets used by the main project README.

Current files:

- `hero-preview.gif`
- `hero-preview.png`
- `setup-preview.png`
- `predict-preview.png`
- `feedback-preview.png`

These are real screenshots and one short animated hero captured from the local browser demo.

If you later capture real product media, the cleanest replacement path is:

1. Keep the same filenames if you want the main README to update automatically.
2. Replace `*.png` with fresher screenshots or an animated GIF.
3. Preserve roughly similar aspect ratios so the README layout stays balanced.

Recommended real captures:

- keep the UI zoom consistent across captures
