# bcd v0.1.0

Initial public release of bcd, a local-first personal choice mirror powered by Codex CLI.

## Highlights

- Local profile and decision memories stored under `~/.bcd`
- React/Vite UI for onboarding, decisions, focused result review, profile review, and memory browsing
- Node API for Codex checks, option suggestions, prediction, feedback, and memory creation
- Codex-backed option suggestions that append to existing options
- Merged memory selection and prediction to reduce Codex CLI round trips
- Fast prediction mode with adaptive deep-mode escalation
- Three-role deep prediction panel for value fit, practicality, and regret risk
- Prediction prompts consider the user's wording as a weak, grounded preference signal
- Markdown memory cards with frontmatter
- MIT license and public contribution docs

## Install

```bash
git clone https://github.com/ronut01/bcd.git
cd bcd
npm install
npm run dev
```

Open [http://127.0.0.1:5173](http://127.0.0.1:5173).

## Verify

```bash
npm run typecheck
npm test
npm run build
```

## Notes

- Requires an authenticated local Codex CLI for real predictions.
- Runtime data is stored outside the repository under `~/.bcd`.
- This release intentionally does not include a local fallback prediction engine.
