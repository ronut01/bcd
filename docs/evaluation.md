# Evaluation

The repository includes a lightweight evaluation script for sample decision cases.

## Initial metrics

- top-1 accuracy
- average confidence assigned to the top prediction
- per-category case breakdown

## Purpose

The MVP evaluation is not meant to be a final benchmark. It is a reproducible sanity check that:

- the end-to-end flow works
- the baseline predictor produces stable outputs
- future retrieval or ranking upgrades can be compared against a known baseline
